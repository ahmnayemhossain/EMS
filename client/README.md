
  # High-Fidelity Design System

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  If PowerShell blocks `npm` scripts (execution policy), run via CMD:
  `cmd /c "npm run dev"`.
